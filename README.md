# python_click_bot_WOW-EmuHack
Click bot that helps you optimize the use of Emu hack for World if Warcraft(2.4.3) while you are away.

To use it you need:

1. WoWEmuHacker5 
2. Basic knowedge of your surrounding in the game. 
3. Propper key bindings for the actions you want. 

The simple functions in the clickbot.py are made with discriptions so that you know what every single one do and later
after line 70 (while Loop: ) you can arrange the actions of every function, in or ther that suits you.
